# Titanic Survival Prediction using Machine Learning

## Project Overview
This project predicts whether a passenger survived the Titanic disaster using machine learning techniques. The model analyzes passenger information such as age, gender, ticket class, and fare to determine survival probability.

## Objective
The objective of this project is to build a classification model that predicts whether a passenger survived or not based on historical data.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset
The dataset contains passenger details from the Titanic disaster, including:

- PassengerId
- Pclass (Passenger Class)
- Sex
- Age
- SibSp (Number of siblings/spouses aboard)
- Parch (Number of parents/children aboard)
- Fare
- Embarked
- Survived (Target variable)

## Machine Learning Model
This project uses classification algorithms such as:

- Logistic Regression
- Decision Tree
- Random Forest

## Steps in the Project
1. Importing required libraries
2. Loading the dataset
3. Data preprocessing
4. Handling missing values
5. Feature encoding
6. Feature scaling
7. Training the classification model
8. Evaluating model performance

## Output
The model predicts whether a passenger survived (1) or did not survive (0).

## Evaluation Metrics
The model performance is evaluated using:

- Accuracy
- Confusion Matrix
- Classification Report

## Conclusion
Machine learning models can analyze historical data and identify patterns that influence survival outcomes. This project demonstrates the use of classification techniques in predictive analytics.
